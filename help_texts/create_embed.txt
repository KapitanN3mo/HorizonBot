Данные в команду передаются в формате JSON
                     **Аргументы:**
**channel**: Канал через решётку
                      **Данные**
:one:-**channel**: ID канала куда будет отправлен embed **ОБЯЗАТЕЛЬНЫЙ АРГУМЕНТ**
:two:-**title**: Заголовок embed **ОБЯЗАТЕЛЬНЫЙ АРГУМЕНТ**
:three:-**text**: Текст рамки. Может быть в несколько строк **ОБЯЗАТЕЛЬНЫЙ АРГУМЕНТ**
:four:-**color**: Цвет рамки. В формате 0x(hex) **ОБЯЗАТЕЛЬНЫЙ АРГУМЕНТ**
:five:-**show_author**: Указывает, отображать ли автора рамки. True,true - да, иначе нет **ОБЯЗАТЕЛЬНЫЙ АРГУМЕНТ**
:six:-**footer**: Нижняя подпись **НЕОБЯЗАТЕЛЬНЫЙ АРГУМЕНТ**
**ЕСЛИ В ТЕКСТЕ ИСПОЛЬЗУЮТСЯ КАВЫЧКИ ИХ НЕОБХОДИМО ЭКРАНИРОВАТЬ С ПОМОЩЬЮ ТРЁХ ОБРАТНЫХ СЛЕШЕЙ**
